# Haskell - functional programming

## ENG
Haskell is a simple language used primarily for learning functional programming.
During the classes, I learnt about creating recursive functions, list operations, type evaluation, creating binary trees, defining new types
and exploring interesting algorithms such as Karatsuba’s algorithm and the ‘divide and conquer’ algorithm.

## PL
Haskell to prosty język, który służy głównie do nauki programowania funkcyjnego.
Podczas zajęć poznałem tworzenie funkcji rekurencyjnych, operacji na listach, ewaluacji typów, tworzenia drzew binarnych, tworzenia nowych typów 
czy poznanie ciekawych algorytmów takich jak Karatsuba czy algorytm "dziel i rządź".
